開き方
1. ZIPを解凍
2. index.html をダブルクリック
3. ブラウザで利用

・43試合（マッチNo.1〜45、欠番5・8）
・試合一覧
・順位表
・得点／アシスト／G+Aランキング
・チーム別成績
・試合追加

・現版は試合結果・得点・アシストを中心に反映しています。

Chugoku Football Data v1.0 RC 導入ガイド

1. サイト本体

ZIPの中身をGitHub Desktopのリポジトリフォルダへ上書きし、Commit → Push originします。

2. Actions確認

GitHubのCode画面に .github が表示されれば成功です。表示されない場合は INSTALL_GITHUB_ACTIONS.command をダブルクリックしてください。

3. 今すぐ自動取得を試す

GitHub → Actions → Sync official match results → Run workflow。

4. 結果確認

緑のチェックなら成功。赤い×なら、その実行画面のスクリーンショットを送ってください。
